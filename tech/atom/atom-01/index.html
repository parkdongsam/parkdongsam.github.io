    Connect GitHub with Atom¶

    Sign Up GitHub¶

    1. GitHub 홈페이지에 접속합니다.
    2. 회원 가입을 위해 Sign Up 버튼을 클릭합니다.
    3. Login 시 사용할 Username과 Email Address, Password를 입력한 후, 하단 퀴즈를 진행합니다.
    4. 모든 정보를 입력하고 Create Account 버튼을 클릭합니다.
    5. 계정 검증을 위해 다시한번 퀴즈를 풀고, Join a free plan 버튼을 클릭합니다.
    6. 메일이 도착하면 Verify email address 버튼을 클릭합니다.

    Install Git¶

    1. Git 홈페이지에 접속합니다.
    2. Downloads 메뉴나 좌측 모니터 모양의 메뉴에서 설치파일을 다운로드 합니다. (약 45MB)
    3. 다운로드가완료되면, 설치를 진행합니다.

    Create Repository¶

    1. 우측 상단 프로필 아이콘을 클릭한 후, Your Repository를 클릭합니다.
    2. 초기 화면으로 이동되면, 좌측 New버튼을 클릭합니다.
    3. Repository name을 입력한 후, Create repository버튼을 클릭합니다.
    4. Repository를 Public혹은 Private하게 사용할 것인지를 선택할 수 있습니다.

    Set Atom¶

    1. Atom에서 Ctrl + , 키를 입력하여 Setting 탭을 엽니다.
    2. Install 메뉴를 선택한 후, git-plus 패키지를 검색하여 Install 버튼을 클릭합니다.
    3. Ctrl + Shift + A 키를 이용하여 프로젝트 폴더를 생성하거나 지정합니다.
    4. 좌측 Project 탭에서 생성한 폴더를 클릭 한 후, a키를 입력하면 새로운 파일을 만들 수 있습니다.
    5. Ctrl + s 키를 입력하여 저장한 후, Ctrl + Shift + 9 키를 이용하여 Git Tab을 엽니다.
    6. Git Tab에 있는 Create Repository 버튼을 클릭한 후, 현재 프로젝트 폴더를 지정합니다.
    7. Init 버튼을 클릭하면, 프로젝트 폴더 아래 .git 폴더가 생성됩니다.
    8. 해당 과정은 $git init 이라는 명령어로 대체 할 수 있습니다.

    Use Git¶

    1. Git Bash를 실행한 후, 프로젝트 폴더로 이동합니다. (Local Repository)

    2. git push 명령어 입력 시 최초 한번 로그인이 필요합니다.

      git config user.name [USER_NAME]
git config user.email [EMAIL]
git remote add origin https://github.com/[USERNAME]/[REPOSITORY_NAME].git

      git add README.md
git commit –m “First commit”
git push –u origin master

    Test¶

    1. GitHub Repository에서 Push한 README.md 파일을 확인할 수 있습니다.
    2. Atom에서 README.md 파일의 내용을 변경한 후, 파일을 저장하면 Unstaged Changes에 파일이 위치합니다.
    3. Stage All 을 클릭하여 모든 파일을 Staged Changes로 이동시킬 수 있습니다.
    4. 파일을 각각 클릭하여 개별적으로 이동 시킬 수 있습니다.
    5. Commit message에 내용을 작성하면, 하단에 있는 Commit to master 버튼이 활성화 됩니다.
    6. Commit to master 버튼을 클릭하면 하단에 Push 버튼이 활성화 되고 Repository로 Push 할 수 있습니다.
